# 10人“奇迹商人”（网易版）规则锁定

## 阵容
- 2狼人 + 1狼王 + 1预言家 + 1女巫 + 1奇迹商人 + 4村民
- 带警长
- 胜利条件：屠边

## 夜间顺序（建议时序）
1. 狼人行动（刀人）
2. 女巫行动（救/毒）
3. 预言家验人
4. 奇迹商人最后睁眼（仅一次赠礼，不能给自己）
   - 可赠：预言家验人 / 女巫毒 / 护盾
   - 当夜生效
   - 护盾：挡刀一次（一次性）

## 白天规则
- 警长竞选、发言、投票
- 商人若赠礼给狼人：首轮警长发言投票后，商人公开暴毙
- 狼王白天被放逐可开枪
- 狼王夜里被刀：天亮可开枪
- 狼王被毒：不可开枪

## 女巫规则
- 首夜可以自救
- 第二夜不可以自救
- 赠礼“毒”：仅一次毒人技能，不含救人

## 预言家与赠礼验人
- 预言家与赠礼“验人”对象可以同夜使用

## 警长规则
- 投票前放手即退选
- 跳预言家必须报警徽流（常留防爆）

## 说明
- 本板子无固定“幸运儿”角色
- “幸运儿”仅作为话术声明，不属于角色体系
